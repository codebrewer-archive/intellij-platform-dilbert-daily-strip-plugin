Planned Features
Possible Features

Saving Downloaded Strips

Some work has been done (but omitted from the current release) to enable saving of downloaded strips.

Making the Plug-in Extensible

Some work has been done (but omitted from the current release) to enable the plug-in to provide extension points so that this plug-in or those written by other authors can provide their own viewers. Combined with access to saved strips this would allow, for example, Weekly Strip Digest, Random Strips and Strip Browser plug-ins to be written.

Taking Advantage of the dilbert.com Browse Feature

URLs of the form http://www.dilbert.com/fast/yyyy-MM-dd/ (to use a java.textSimpleDateFormat pattern) give direct access to the strip of the given date. A way to browse from the plug-in would be useful.
